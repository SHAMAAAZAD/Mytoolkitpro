📋 Loan Details
💳 Loan Pre-Payment
Pre-payment reduces your principal. You can either reduce EMI (same tenure) or reduce tenure (same EMI).
📈 Variable Interest Rate
Add interest rate change periods. The EMI and schedule will recalculate accordingly.
⏸️ Moratorium Period
During moratorium, EMI is paused but interest continues to accrue. This extends loan tenure.
📋 Payment History
All EMI payments and un-payments with date/time and confirmation codes.
🔥 Firebase Setup — Real-time Sync
Steps:
1. console.firebase.google.com → New Project
2. Build → Realtime Database → Create Database
3. Rules: { "rules": { ".read": true, ".write": true } }
4. Project Settings → General → Web App → Config copy karo
5. Niche values paste karo aur Save karo
Config browser ke localStorage mein save hoga. Har device pe alag se setup karna hoga ya HTML file mein seedha likhein.
